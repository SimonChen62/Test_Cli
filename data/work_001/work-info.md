# work_001 作品信息

## 基本信息

- 标题：赵孟頫《行书光福重建塔记卷》中段局部
- 书体：行书（亦常称行楷）
- 作者：赵孟頫
- 年代：元至治元年（1321）
- 材质与形制：纸本墨笔，手卷
- 尺寸：纵 28.6 厘米，横 284 厘米
- 收藏：上海博物馆
- 官方资料：<https://www.shanghaimuseum.net/mu/frontend/pg/m/article/id/CI00000601>
- 来源说明：作品信息依据上海博物馆；当前展示图片的直接来源与授权仍需项目组核实
- 裁剪说明：当前 V1 使用整幅横向作品的中段局部，原图已归档到本地 `data/source/Zmf_full.jpg`，不建议直接上传 Git

## 使用说明

请将裁剪后的作品图片保存为：

```text
data/work_001/original.png
```

图片建议：

- 尽量正向、清晰、无强烈透视变形
- 背景和墨迹对比明确
- 第一版 Demo 优先选择局部，不要一次放入过长作品

## 当前裁切判断

原图是非常宽的横向长卷。如果直接用于 V1，网页显示会过窄，标注点也很难点击。因此当前选用中段局部作为 `original.png`。

这样做的理由：

- 局部已经能体现字列之间的气脉、留白和笔墨轻重线索
- 对普通观众更容易观察
- 对网页交互和人工标注更稳定
- 后续如果研究整体章法，可以再增加整卷浏览或多局部对比
