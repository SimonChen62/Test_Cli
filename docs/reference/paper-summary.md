# 论文参考摘要：Seeing Qi and Void

参考 PDF：`docs/reference/seeing-qi-and-void-callilens.pdf`

## 项目定位

CalliLens 不是书法识别、生成或临摹系统，而是一个帮助初学者理解书法审美概念的交互脚手架。它把“气脉”“虚实”等抽象概念锚定到可见的笔画、墨色、留白和空间节奏上。

## 核心框架

论文草案强调：

```text
Formal Features -> Perceptual Experience -> Aesthetic Concepts -> Reflective Articulation
形式特征 -> 感知经验 -> 审美概念 -> 反思表达
```

这与当前 Demo 的三段解释对应：

- `formal` 对应形式特征
- `perception` 对应感知经验
- `aesthetic` 对应审美概念

第一版 Demo 暂不做反思任务，但后续可以加入“用户指出证据”和“用户写下自己的解释”。

## 对 V1 的启发

- 先保留原作观看，再打开解释图层
- 可视化层必须可开关，避免过度解释压住原作
- 气脉用动态路径表达运动趋势
- 虚实用留白高亮表达空间参与结构
- 技术只辅助观察，不替代审美解释
