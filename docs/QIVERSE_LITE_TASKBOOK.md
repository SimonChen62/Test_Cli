# CalliLens + QiVerse 电影体验分支任务书

## 1. 分支目标

当前分支 `codex/qiverse-cinematic-flow` 用来验证一个更适合课堂展示的 QiVerse 方向：不再把老师文档做成干巴巴的 XR 工程样板，而是先做一个前端电影感体验。

核心效果：

```text
点击开始
→ 星河视角出现散落星点
→ 星点慢慢聚成赵孟頫《光福重建塔记》墨迹长卷
→ 镜头进入墨迹内部遨游
→ 淡金粒子提示“形断势连”
→ 空白区域显现
→ 回到原作
```

该分支不替换主系统。CalliLens 继续负责作品库、平滑浮雕 3D、本地 RAG、管理员上传、用户登录和反思记录；QiVerse 作为独立入口验证沉浸式体验是否更有表现力。

## 2. 为什么改成电影式前端体验

老师文档强调的不是“在 VR 里看书法展”，而是让用户经历：

1. Silence / 静墨
2. Enter the Ink / 入墨
3. Ride the Stroke / 御笔而行
4. Follow the Qi / 追势
5. Walk into the Void / 入白
6. Return to Original / 回看原作

之前的 XR 原型更像技术验证：线条、盒子、状态机都有，但观感弱，不适合第一眼演示。新的方向把它包装成一个短视频式、可交互的 Three.js 场景，先用视觉冲击把“进入书法内部”这件事讲明白。

## 4. 技术实现

### 4.1 原图采样成点云

前端读取：

```text
data/work_003/original.png
```

然后用浏览器 Canvas 对原图进行缩放采样：

```text
原图
→ Canvas drawImage
→ 读取像素 RGB
→ 根据 darkness 判断墨迹区域
→ 选取深色像素
→ 生成 Three.js Points 点云
```

墨色越深，点云的 Z 轴高度越高；浅墨、飞白和细线区域高度更低。这里仍然是“可见图像特征转空间效果”，不是恢复真实笔顺。

### 4.2 星河到长卷

每个墨迹点有三套坐标：

```text
galaxy：星河散点位置
target：原作墨迹位置
scatter：入白时的散开位置
```

不同章节通过插值切换坐标，所以用户会看到星点从宇宙慢慢收束成书法长卷。

### 4.3 入墨和御笔

入墨阶段拉近相机，并提高墨迹点的 Z 轴。御笔阶段让相机沿长卷方向移动，形成“在墨迹内部穿行”的感觉。

答辩表述必须保持边界：

> 我们没有声称恢复赵孟頫真实笔顺。当前版本是基于图像墨迹点云和解释性路径的空间体验，用来帮助观众感知线条运动感。

### 4.4 追势和入白

`qiverse/calligraphy-work.json` 中保留少量解释性数据：

```text
qiLinks：淡金色粒子路径，用来提示形断势连
voidRegions：字内、字间、行间留白区域
```

这些数据是人工解释性标注，不是算法自动判断“气韵”。

## 5. 和 RAG 的结合

RAG 不负责生成动画，它负责解释动画：

用户可以问：

```text
QiVerse 是什么？
为什么要把星点聚成书法？
Enter the Ink 是什么意思？
Follow the Qi 是不是算法判断气韵？
Walk into the Void 为什么重要？
```

系统应基于本地知识库回答，并明确：

1. QiVerse 是体验层，不是书法评分系统。
2. 星点来自原作图像的墨迹采样。
3. 气脉和留白是解释性可视化。
4. 不恢复真实笔顺。
5. 不判断真伪和书法水平。

## 6. 测试路线

启动：

```powershell
.\start-demo.ps1
```

打开：

```text
http://127.0.0.1:5190/qiverse/
```

测试：

1. 页面能看到“墨气星河”开始面板。
2. 点击“开始入墨”，星河开始播放。
3. 点击“成字”，星点能收束成《光福重建塔记》长卷。
4. 点击“御笔”，镜头进入点云内部移动。
5. 点击“追势”，能看到淡金粒子路径。
6. 点击“入白”，墨迹淡出，留白区域显现。
7. 点击“回看”，回到原作长卷。
8. 控制台无阻塞错误。

已执行检查：

```powershell
node --check qiverse\qiverse.js
```

并用 Playwright 截图确认页面可渲染、章节可切换、无 console error。

## 7. 后续可做

如果老师认可这个方向，再做：

1. 在主 CalliLens 页面强化 QiVerse 入口视觉。
2. 增加 60-90 秒自动演示模式。
3. 增加字幕导览和轻音乐，但保留静音可用。
4. 将 `qiLinks`、`voidRegions` 做成管理员可视化标注工具。
5. 再考虑 WebXR / Meta Quest，而不是现在就强行做完整 VR。

## 8. 当前边界

当前不做：

1. 不自动识别真实笔顺。
2. 不自动判断气韵。
3. 不给书法水平评分。
4. 不判断真伪。
5. 不把随机粒子当作学术分析。

推荐答辩一句话：

> QiVerse 电影体验把《光福重建塔记》的可见墨迹采样成点云，让星点从宇宙收束为书法长卷，再通过入墨、御笔、追势和入白的空间镜头，帮助观众从运动、气脉和留白角度重新观看原作。
