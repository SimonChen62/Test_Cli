# CalliLens 人工标注指南

## 总原则

人工标注不是凭空写美学解释，而是根据原图和 OpenCV 辅助图判断。每个标注都必须包含三段：

- `formal`：形式描述，画面上客观能看到什么
- `perception`：感知经验，普通用户看到后可能产生什么感受
- `aesthetic`：美学解释，这个观察如何帮助理解气脉、虚实或笔墨节奏

不要写：

- 系统检测到真实力道
- 系统判断这幅字气韵很好
- 模型识别出笔顺
- 自动审美评分

可以写：

- 辅助图提示笔画粗细变化
- 骨架图帮助观察方向承接
- 留白候选图帮助定位空白区域
- 人工标注认为这里体现形断势连

## 气脉路径

主要看：

- 原图
- `skeleton.png`
- `binary.png`
- `stroke_width.png`

满足以下任意 2 条，可以标为气脉候选；满足 3 条以上，优先标注：

| 指标 | 判断方式 | 推荐标签 |
|---|---|---|
| 方向承接 | 上一笔收笔方向指向下一笔起笔方向 | 顺势 |
| 距离接近 | 两笔距离较近，视觉上容易形成连续关系 | 连贯 |
| 转折牵引 | 笔画有明显转向、回带、牵引 | 转势 |
| 路径连续 | 多个笔画能形成连续观看路径 | 行气 |
| 形断势连 | 墨迹断开，但视觉趋势继续 | 形断势连 |

## 虚实留白

主要看：

- 原图
- `binary.png`
- `void_candidates.png`
- `ink_density.png`

满足以下任意 2 条，可以标为虚实候选；满足 3 条以上，优先标注：

| 指标 | 判断方式 | 推荐标签 |
|---|---|---|
| 密旁有空 | 密集笔画旁边有明显空白 | 密处得空 |
| 字内空白 | 字内部被笔画围出空间 | 字内留白 |
| 字间留白 | 字与字之间形成呼吸距离 | 字间呼吸 |
| 疏密对比 | 一边密、一边空，形成视觉平衡 | 疏密平衡 |
| 飞白 | 笔画内部出现白色断裂，实中有虚 | 实中有虚 |
| 空白导向 | 空白影响视线流动或结构稳定 | 空处有势 |

## 笔墨轻重线索

统一称为“笔墨轻重线索”，不要称为“力道检测”。

主要看：

- 原图
- `stroke_width.png`
- `ink_density.png`
- `skeleton.png`

| 指标 | 判断方式 | 推荐标签 |
|---|---|---|
| 粗到细 | 笔画从重到轻，可能提示提笔 | 由按转提 |
| 细到粗 | 笔画从轻到重，可能提示按笔 | 由提转按 |
| 转折变粗 | 转折处粗重，可能有停顿 | 顿挫 |
| 收笔变细 | 收尾变轻，可能形成连贯或结束 | 收势 |
| 墨色变淡 | 可能提示速度变快、墨量减少或收尾 | 枯淡 |
| 墨色变浓 | 可能提示起笔、加重或速度放慢 | 浓重 |
| 飞白出现 | 可能提示干笔、速度或摩擦 | 飞白 |
