# CalliLens 课堂演示脚本

这份脚本按实际点击顺序写，适合课堂展示或答辩时照着走。

## 1. 启动项目

在项目根目录运行：

```powershell
python -m pip install -r requirements.txt
.\start-demo.ps1
```

打开：

```text
http://127.0.0.1:5190/web/
```

如果一键启动失败，手动开两个 PowerShell：

```powershell
python -m uvicorn backend.app.main:app --reload --port 8000
python -m http.server 5190 --bind 127.0.0.1
```

## 2. 首页说明

先说明项目定位：

> CalliLens 不是自动评价书法好坏，而是把赵孟頫《光福重建塔记》做成数字导览：原图浏览、OpenCV 墨迹处理、Three.js 悬浮 3D 和本地 RAG 问答。

点击“网页已存库”。

## 3. 进入默认作品

选择：

```text
赵孟頫《光福重建塔记》
```

说明这件作品的基础事实：

- 作者：赵孟頫
- 时代：元代
- 时间：至治元年，1321 年
- 书体：行书
- 馆藏：上海博物馆

## 4. 第一印象

在三个输入框里填写简短内容：

```text
整体安静而有流动感。
长卷中部笔画连带处运动感明显。
题名和密集字群附近较紧，留白处较疏。
```

点击“进入观察”。

## 5. 展示 3D 悬浮墨迹

点击图层里的：

```text
3D悬浮
```

演示：

- 点击 `+` 放大
- 点击 `-` 缩小
- 拖动“水平”滑条移动长卷
- 鼠标拖拽画布旋转视角
- 点击“左 / 正 / 右”切换视角

说明：

> 这里不是恢复真实笔顺，而是把图像中可见墨迹深浅转换为空间高度。重墨区域更高，飞白和细线更低，所以侧面看会有悬浮厚度。

## 6. 展示 RAG 问答

在右侧“本地 RAG 导览”输入：

```text
为什么需要 RAG？
```

点击“检索并回答”。

说明：

> 系统先检索本地知识库，再组织回答，并显示来源。没有 API key 也能运行；如果管理员配置了 API，大模型只负责润色，不负责凭空补充事实。

再测试：

```text
赵孟頫是谁？
OpenCV 在项目中做了什么？
Three.js 在项目中做了什么？
没有 API key 能不能运行？
```

## 7. 管理员后台

返回首页，点击“上传个性化书法作品”，输入演示口令：

```text
callilens-admin
```

说明：

> 第一版只开放管理员上传，避免普通用户上传低质量图片导致处理失败。管理员上传后，系统会保存原图、生成 mask、height、缩略图和知识库片段。

## 8. 结束语

最后总结：

> 当前版本先做稳定可演示的 Desktop Demo。老师文档中的 InkVerse / WebXR 会作为第二阶段扩展，接在现有 OpenCV、Three.js 和 RAG 基础上继续做沉浸式体验。
