# 本地 RAG 技术说明
source: 项目技术说明

## RAG 是什么
RAG 是检索增强生成。用户提问后，系统先从本地知识库检索相关资料，再根据资料组织回答。

## 为什么不用 AI 直接回答
直接问 AI 容易出现没有依据的发挥。RAG 可以把回答限制在作品资料、术语解释和项目技术说明范围内，并显示来源。

## 没有 API key 能不能运行
可以。第一版使用本地检索和模板化回答，不依赖付费 API。有 API key 时，大模型只负责润色表达，不负责凭空补充事实。

## 管理员上传与知识库
管理员上传作品时，需要填写作品名称、作者、年代、简介、背景和来源。系统会把这些文字资料写入对应作品的知识库，使新作品也能被检索问答。
