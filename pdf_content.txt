2. Motivation
中国书法不仅是文字书写，也是一种高度凝练的视觉艺术。其审美价值并不只存在于字形是否端
正、作品是否“好看” ，而是体现在笔画运动、墨色变化、粗细轻重、留白关系、行气章法和整体
节奏之中。对于书法专家来说，“气脉”“ 虚实”“ 行气”“ 飞白”“ 章法”等概念可以从作品形式中被感
知；但对于非专家观众而言，这些概念往往停留在抽象术语层面，很难从具体作品中指出其视觉
依据。
现有数字书法研究和系统多关注几个方向：书法临摹训练、VR 书写体验、书写过程捕捉、字体
生成、风格识别或笔画分类。这些工作有助于“写得像” 或“识别得准” ，但较少关注一个更基础的
欣赏问题： 普通观众如何学会看书法，如何从可见的笔墨形式进入抽象的中国美学理解。你原
proposal 里已经明确指出，这个项目的核心不是做一个普通书法学习软件，而是把书法中的笔
画、墨色、结构、章法、留白和节奏，转化为用户可理解、可感知、可交互探索的中国美学思
想。
因此，本研究提出 CalliLens：一个面向书法初学者的交互式与空间化可视化系统。它通过动态
气脉流线、虚实切换、留白高亮、3D 笔势路径和反思任务，引导用户从“看见笔画” 进一步走向
“理解笔画如何产生气、虚、节奏和美感” 。
3. Research Gap
这个项目最需要强调的 gap 不是“现有系统没有用 AR/VR/3D” ，而是：
现有系统较少研究如何把抽象的书法美学概念转化为非专家用户可以观察、理解和表达的
交互体验。
可以具体分成三层。
Gap 1：Skill learning 多，aesthetic understanding
少
很多书法系统关注如何临摹、如何写字、如何捕捉书写动作、如何生成某种风格字体。但本项目
关注的是：

=== Page 3 ===
用户如何欣赏书法，而不是如何书写书法。
这是从 skill learning 到 aesthetic understanding 的转向。
Gap 2：形式识别多，形式—思想连接少
现有技术可以提取笔画、骨架、粗细、墨色、字形结构等特征。但对于书法欣赏来说，真正重要
的是：
这些形式特征如何让用户理解“气脉”“ 虚实”“ 行气”“ 节奏”等审美概念。
你的 proposal 中提出的核心框架非常重要：
Brushwork Features → Perceptual Experience → Aesthetic Thought
笔墨形式特征 → 感知经验 → 美学思想。
这个框架应该成为 proposal 的理论主线。
Gap 3：解释容易过度，文化审美需要保留开放性
书法欣赏不是机械知识灌输。系统如果过度标注、过度解释，可能会遮蔽原作的审美体验。因此
CHI reviewer 会关心：
技术是否真的帮助理解？
是否尊重原作？
是否允许用户形成自己的感受？
是否把文化概念说得太死？
所以本项目不仅研究“如何可视化” ，也研究：
如何在引导理解和保留审美开放性之间取得平衡。

6. System Design: CalliLens
6.1 Design Goals
从 formative study 和专家访谈中，可以形成以下设计目标。
DG1: Anchor abstract concepts in visible evidence
系统必须把“气脉”“ 虚实”等抽象概念锚定到具体视觉证据上，例如笔画方向、粗细变化、留白区
域、字距行距。
DG2: Support layered viewing
用户应能在原作、标注、动态流线、3D 解释之间自由切换，避免一开始就被系统解释压住。
DG3: Preserve aesthetic openness
所有可视化层都应可开关、可调透明度，并保留纯观看模式。
DG4: Enable reflective articulation
系统不只是展示解释，还要让用户回答、标注、比较和反思。
DG5: Use spatial visualization only when it clarifies
invisible processes
3D、AR 、VR 不作为炫技，而是用于解释静态图像中难以感知的运动、节奏和生成过程。

7. Main Interaction Modules
Module 1: Original Viewing Mode
用户先看原作，不显示任何标注。
系统先问：
“你觉得这幅作品给你的整体感觉是什么？”
“你认为哪里最有运动感？”
“你觉得哪些地方最疏朗或最紧密？”
这样做的意义是：
先保留用户自己的直觉，再进入系统解释。
Module 2: Qi Flow View
这是最核心模块。
用户打开 Qi Flow V iew 后，系统在作品上显示动态流线，表示：
笔画方向；
起笔、行笔、转折、收笔；
上一笔与下一笔之间的势；
字与字之间的行气；
行与行之间的章法推进。
你的 proposal 已经把 Qi Flow V iew 定义为最重要模块，用来把“气” 从抽象词转化为可观察的动
态关系。
系统解释示例：
这两笔之间没有实际连线，但第一笔的收势向右下，下一笔从相近方向起势，因此形成“形

断而势连”。这里的气脉不是线条必须连在一起，而是运动趋势和精神气息的连续。
用户任务：
“请指出另一处你认为存在‘ 形断势连’ 的地方。”
“请解释为什么这里的气脉没有断。”
Module 3: Void-Solid View
用户可以切换三种视图：
View A: Solid
只看墨迹。
View B: Void
弱化墨迹，突出留白、字距、行距、空隙。
View C: Void-Solid Relation
同时显示墨迹和空白，展示两者如何互相组织。
你的 proposal 中也强调，虚实模式可以帮助用户发现空白不是没有内容，而是在组织笔画、调
节节奏、形成呼吸。
系统解释示例：
这一处笔画较密，但右侧留出较大空白，使密处不显拥堵。虚处让实处得以呼吸，形成“虚
实相生”。
用户任务：
“请找出一处你认为空白参与结构的地方。”
“如果去掉这块留白，整幅作品的节奏会有什么变化？”

Module 4: 3D Brushwork Path View
这个模块用于解释“笔画是运动过程的痕迹” 。
系统将关键笔画转成 3D ribbon 或 tube ：
3D 表达 对应书法意义
路径方向 笔势
管道粗细 提按、轻重
透明度/ 材质 墨色浓淡
断裂纹理 飞白
时间轴播放 行笔过程
弯曲程度 转折、顿挫
局部停顿 节奏变化
你的 proposal 已经提出可用 Three.js 把笔画路径变成 3D tube/ribbon ，用路径方向表示笔势，
用半径表示粗细/ 压力，用透明度表示墨色浓淡，用断裂纹理表示飞白。
但这里要注意表述：
不要说“复原王羲之真实书写过程” 。
要说：
expert-informed interpretive reconstruction
基于专家标注和图像特征的解释性重建。
这样更严谨。
Module 5: AR Overlay Mode
AR 可以作为博物馆或课堂场景的扩展模块。

9.2 Participants
建议：
24–36 位非专家用户。

Task 3: Evidence-based Aesthetic Interpretation
让用户回答：

让用户看一幅没有系统解释的新作品，要求他们独立分析。

15. Revised Abstract
可以直接用这个版本：
Chinese calligraphy embodies aesthetic concepts such as qi-flow and void-solid interplay
through subtle formal features, including brush movement, ink variation, rhythm, spatial
layout, and negative space. However, novice viewers often lack the visual literacy needed
to connect these visible features with abstract aesthetic ideas. We present CalliLens, an
interactive and spatial visualization system that scaffolds novice viewers’ aesthetic
understanding of Chinese calligraphy. Based on expert-informed annotations, CalliLens
provides coordinated views including qi-flow animation, void-solid exploration, and 3D
brushwork path visualization. Rather than teaching users how to write calligraphy, the
system guides them to observe, interpret, and reflect on how aesthetic meaning emerges
from calligraphic form. We propose a Brushwork-to-Aesthetic Understanding framework
and evaluate CalliLens through a user study comparing traditional text explanation, static
annotation, and interactive visualization. Our work contributes empirical insights into
novice calligraphy appreciation, an interactive visualization prototype, and design
implications for culturally situated visualization of intangible aesthetic knowledge.
